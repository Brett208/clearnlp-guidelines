# Semantic Role Labels 

## Numbered Arguments

| Label | Description |
|:-----:|:------------|
| A0   | Agent |
| A1   | Patient |
| A2   | Instrument, benefactive, attribute |
| A3   | Starting point |
| A4   | Ending point |
| AA   | External causer |

## Modifiers

| Label | Description |
|:-----:|:------------|
| ADJ   | Adjectival (for nominal predicates)|
| ADV   | Adverbial |
| CAU   | Cause |
| COM   | Comitative |
| DIR   | Direction |
| DIS   | Discourse |
| DSP   | Direct speech |
| EXT   | Extent |
| GOL   | Goal |
| LOC   | Location |
| MNR   | Manner |
| MOD   | Modal |
| NEG   | Negation |
| PRD   | Secondary predication |
| PRP   | Purpose |
| PRR   | Light verb predicate |
| REC   | Recipricol |
| TMP   | Temporal |
